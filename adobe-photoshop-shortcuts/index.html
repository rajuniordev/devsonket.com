ফাইল (File)

  • নতুন ফাইল (New)

    Ctrl + N

  • ফাইল ওপেন করতে (Open)

    Ctrl + O

  • ফাইল এডোবি ব্রিজে ব্রাউজ করতে (Browse in Bridge)

    Alt + Ctrl + O or Shift + Ctrl + O

  • ফাইল ওপেন হিসাবে (Open As)

    Alt + Shift + Ctrl + O

  • ফাইল বন্ধ করা (Close)

    Ctrl + W

  • সকল ফাইল বন্ধ করতে (Close All)

    Alt + Ctrl + W

  • ফাইল বন্ধ করুন এবং এডোবি ব্রিজে যান (Close and Go to Bridge)

    Shift + Ctrl + W

  • ফাইল সেভ করতে (Save)

    Ctrl + S

  • ফাইল ফরম্যাট হিসেবে সেভ করতে (Save As)

    Shift + Ctrl + S or Alt + Ctrl + S

  • ওয়েব এর জন্য ফাইল সেভ করতে (Save for Web)

    Alt + Shift + Ctrl + S

  • ফাইল আগেকার অবস্থায় আনতে (Revert)

    F12

  • ফাইল সম্পর্কিত তথ্য (File Info)

    Alt + Shift + Ctrl + I

  • ফাইল প্রিন্ট করতে (Print)

    Ctrl + P

  • এক কপি ফাইল প্রিন্ট করতে (Print One Copy)

    Alt + Shift + Ctrl + P

  • ফাইল থেকে বের হতে (Exit)

    Ctrl + Q

  • ছবি সিলেক্ট এবং মাস্কিং করা

    Ctrl + Alt + R

ইমেজ (Image)

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে লেভেল সিলেক্ট কি (Adjustments > Levels)

    Ctrl + L

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে কার্ভস সিলেক্ট কি (Adjustments > Curves)

    Ctrl + M

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে হিউ/স্যাচুরেশন সিলেক্ট কি (Adjustments > Hue/Saturation)

    Ctrl + U

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে কালার ব্যালান্স সিলেক্ট কি (Adjustments > Color Balance)

    Ctrl + B

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে সাদা কালো সিলেক্ট কি (Adjustments > Black & White)

    Alt + Shift + Ctrl + B

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে ইনভার্ট সিলেক্ট কি (Adjustments > Invert)

    Ctrl + I

  • ইমেজ এর কালার কে গ্রেস্কালে কনভার্ট করতে (Desaturate)

    Shift + Ctrl + U

  • ইমেজ এর অটো টোন কি (Auto Tone)

    Shift + Ctrl + L

  • ইমেজ এর অটো কনট্রাস্ট কি (Auto Contrast)

    Alt + Shift + Ctrl + L

  • ইমেজ এর অটো কালার কি (Auto Color)

    Shift + Ctrl + B

  • ইমেজ সাইজ কি (Image Size)

    Alt + Ctrl + I

  • ইমেজ ক্যানভাস কি (Canvas Size)

    Alt + Ctrl + C
